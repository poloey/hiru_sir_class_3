## tag 

* `table`
* `tr`
* `th`
* `td`

* `thead`
* `tbody`
* `tfoot`

* `div` - block level element 
* `span` - inline level element 


## attributes 

* `border`  is used in table 
* `style`  is used in inline style 
* `id` for make a tag unique
* `class` for common selection 


## type of style

* inline style    

~~~php
<h1 style="color: red" >Hello world </h1>
~~~

* internal style 

~~~css
<style>
  h1 {
    color: red;
  }
</style>
~~~

## css selector 

1. tag selector

~~~css
h1 {
  color: red;
}
~~~

here,
`h1` - tag    
`color` - property
`red` - value

2. id selector 

id selector start with `#`

~~~css
#first-heading {
  color: red;
}
~~~

3. class selector

class selector start with `.`

~~~
.hiru-sir-fav-color {
  color: red;
}
~~~
